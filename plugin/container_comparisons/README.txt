-------------------------------------------------------------------
*Steps for compiling this project:
-------------------------------------------------------------------
1. $cmake ./
2. $make


-------------------------------------------------------------------


-------------------------------------------------------------------
*Running the bins
-------------------------------------------------------------------
$sh run.sh (using run script)

or

Manually run both binaries from bin folder in the current directory
-------------------------------------------------------------------


-------------------------------------------------------------------
*Output
-------------------------------------------------------------------
Results after running bins will be in "output_(bin_name).txt" in the current dir
